Get up and running with Dial Code in minutes.

---

## Quick Path

1. **[Install](installation.md)** - Set up Node.js and install Dial Code
2. **[Authenticate](authentication.md)** - Connect to an LLM provider

---

## Prerequisites

- **Node.js 20+** - [Download](https://nodejs.org/)
- **Git** - For cloning the repository
- **Terminal** - Any modern terminal emulator

---

## 30-Second Start

```bash
# Install
git clone https://github.com/dipankar/dial-coder.git
cd dial-coder && npm install && npm install -g .

# Run (opens browser for Qwen OAuth)
dial
```

That's it! Qwen OAuth provides 2,000 free requests per day.
